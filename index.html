Welcome to Request Catcher

This application allows multiple users to catch HTTP requests in their own namespaces.

To get started, please go to /default or create your own unique namespace by going to /your-namespace.

Example: /default

Example calls

default namespace

curl http://CATCHER_URL/default/data?name=John&age=30
curl -X POST http://CATCHER_URL/default/data -H "Content-Type: application/json" -d '{"name":"John", "age":30}'

custom namespace

curl http://CATCHER_URL/your-namespace/data?name=John&age=30
curl -X POST http://CATCHER_URL/your-namespace/data -H "Content-Type: application/json" -d '{"name":"John", "age":30}'
